# 🤖 Spring AI + Ollama Chat Backend

A simple chatbot backend built using **Spring Boot**, **Spring AI**, and **Ollama** that allows you to run and interact with local LLMs like Llama 3.

---

## 🚀 Features

- ⚡ Spring Boot REST API
- 🧠 Spring AI integration
- 🖥️ Local LLM support using Ollama
- 💬 Simple chat endpoint
- 🔌 Easily extendable for frontend or RAG systems

---

## 🛠️ Tech Stack

- Java 17+
- Spring Boot 3+
- Spring AI
- Ollama (Local LLM runtime)
- Maven

---

## 📦 Prerequisites

Make sure you have the following installed:

- Java 17+
- Maven 3+
- Ollama → https://ollama.com
